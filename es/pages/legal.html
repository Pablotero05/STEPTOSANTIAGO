Condiciones de Uso del Sitio Web

El acceso al Sitio Web es gratuito y no requiere suscripción previa, salvo para aquellas secciones que así lo indiquen expresamente. El usuario se compromete a hacer un uso adecuado de los contenidos y servicios del Sitio Web.

El usuario se compromete a:

Hacer un uso lícito del Sitio Web, sin vulnerar la legislación española ni la de la Unión Europea.

No introducir, almacenar ni difundir información falsa, difamatoria, obscena o ilegal.

No vulnerar los derechos de propiedad intelectual o industrial del titular o de terceros.

No utilizar el Sitio Web para enviar comunicaciones comerciales no solicitadas (spam).

No intentar acceder a áreas restringidas del Sitio Web sin autorización expresa.

Queda expresamente prohibido:

Reproducir, distribuir o transformar los contenidos del Sitio Web sin autorización escrita del titular.

Realizar scraping, minería de datos o extracción masiva de contenidos de forma automatizada.

Utilizar el Sitio Web para actividades que puedan dañar la imagen o reputación de Step To Santiago.

Introducir virus, malware o cualquier elemento que pueda dañar los sistemas informáticos del titular o de terceros.
